## 程序运行统计信息

### orb_cv (使用OpenCV ORB)
- 特征提取时间: ~0.012秒
- 特征匹配时间: ~0.0004秒
- 最大匹配距离: 94.0
- 最小匹配距离: 4.0

### orb_self (自实现ORB)
- 特征提取时间: ~0.001秒  
- 特征匹配时间: ~0.0006秒
- 找到的匹配点: 41个

### pose_estimation_2d2d (2D-2D姿态估计)
- 找到的匹配点: 79组
- 计算了基础矩阵、本质矩阵、单应矩阵
- 恢复了旋转矩阵R和平移向量t

### triangulation (三角测量)
- 找到的匹配点: 79组
- 计算了79个3D点的深度信息
- 深度范围: 6.6-12.3 (相对单位)

## 技术细节

- 所有图像均为640x480像素
- 使用TUM数据集的相机参数
- 特征匹配使用汉明距离
- 三角测量基于对极几何约束
